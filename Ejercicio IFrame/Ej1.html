Ejercicio N°1



Preguntas:

  1. Dar un par de ejemplos de uso de etiquetas meta.
  2. Nombrar al menos 2 atributos aplicables a la etiqueta HTML
  3. ¿Como se relaciona un header http con la etiqueta head de HTML?
  4. ¿Que es UTF8?
  5. ¿Que atributo HTML sirve para aplicar estilo, a que etiqueta es aplicable?
  6. ¿Donde se indica que un enlace debe ubicar el documento en una nueva pestaña?
  7. Explicar cuando usarían etiquetas semánticas y con que propósito.
  8. ¿Que atributos de estilo básicos aplicarían a una tabla? Explicar.
  9. ¿Que diferencia existe entre la etiqueta body de un documento HTML y el body del protocolo HTTP?
  10. ¿Que atributos marcan la agrupación de celdas de una tabla?. ¿A que etiquetas se aplican?


Respuestas:

  1. Una etiqueta meta se podría utilizar para recargar la página actual cada una cierta cantidad de segundos, por ejemplo: 
     <meta http-equiv=”refresh” content=”100”/>

    Otro uso podría ser para que el motor de busqueda reciba la descripción de la página, por ejemplo: 
     <meta name="description" content="Ésta página contiene 150 caracteres"/>
  2. Para una etiqueta tipo <table> se podría utilizar el atributo "border" para remarcar los bordes de la tabla / matriz, otro atributo aplicable a todas las etiquetas html es el "id" el cual puede identificar cada etiqueta, de esta forma con una etiqueta de tipo <label> puedo referirme a través del atributo "for" a cualquier otra etiqueta con un "id" particular.
  3. Su relación yace en que aportan data relevante del body de la página.
  4. UTF-8 es un formato de codificación de caracteres Unicode que utiliza símbolos de longitud variable (1 a 4 bytes por caracter Unicode), este formato puede representar cualquier caracter Unicode y los separa en grupos según cuantos bytes se necesitan para representar cada uno de ellos, los que necesitan 1, 2, 3 y 4 bytes.
  5. El atributo que se utiliza para aplicar estilo es el "style" y es aplicable a la etiqueta <img>.
  6. Se indica en el atributo "target" de la etiqueta <a>, seteandolo cómo target="_blank".
  7. Las etiquetas semánticas se utilizan a la hora de marcar una sección en la que se encuentra un tipo de contenido determinado, estas marcan el principio y final de dichos contenidos y podrían ser el contenido principal de un texto, títulos, encabezados, imagenes, enlaces, direcciones, etc.
  8. Se podrían aplicar atributos como "style="border:...;border-collapse:...;background-color:...;width:...;height:...;"" y estos podrían agregarle borde a la tabla, cambiar el color de fondo, aumentar y disminuir el tamaño de las celdas tanto en ancho como en alto, entre otras.
  9. La diferencia está en su contenido, dentro de la etiqueta body de un documento HTML puede haber información con o sin clasificación alguna, quien decide que tipo de información se ve reflejada en la etiqueta es quien escribe el código, en cambio en el body del protocolo HTTP hay información diferente dependiendo de las circunstancias y con qué verbo se llene la primer linea del HTTP, por ejemplo, el contenido del body de un protocolo HTTP enviado al servidor con verbo POST no es el mismo que si se utiliza GET.
  10. La agrupación de celdas en una tabla se puede manejar con los atributos "rowspan" (para agrupar filas) y "colspan" (para agrupar columnas), estos atributos son aplicables a la etiqueta <td> que yace dentro de una etiqueta <table>.
